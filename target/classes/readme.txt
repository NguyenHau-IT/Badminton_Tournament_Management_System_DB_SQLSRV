DESCRIPTION
------------------------------------------------------------
Badminton Event Technology is a desktop application for managing badminton events, including match schedules, scoring, and data storage.
It works fully offline using an embedded H2 Database.

------------------------------------------------------------
FEATURES
------------------------------------------------------------
- Manage players, teams, and events
- Schedule matches and record scores
- Save and load event data locally
- Modern UI with FlatLaf theme
- Works offline without internet

------------------------------------------------------------
INSTALLATION
------------------------------------------------------------
1. Run the installer (EXE or MSI).
2. Follow the on-screen steps to choose installation directory.
3. After installation, launch the app from:
   - Desktop shortcut, or
   - Start Menu > Badminton Event Technology

------------------------------------------------------------
SYSTEM REQUIREMENTS
------------------------------------------------------------
- Windows 10 or later (64-bit)
- At least 1 GB RAM
- 200 MB free disk space

------------------------------------------------------------
HOW TO USE
------------------------------------------------------------
1. Open the application.
2. Load a database from H2 server Sportdata Event Technology for your badminton event.
3. Add players/teams and configure matches.
4. Record match scores as the event progresses.
5. Save your data for future reference.
