Tính năng nổi bật

Giải pháp toàn diện cho giải đấu cầu lông

BTMS cung cấp đầy đủ công cụ để tổ chức và quản lý giải đấu chuyên nghiệp

Quản lý đa sân

Điều khiển đồng thời 5 sân thi đấu với mã PIN riêng biệt. Dễ dàng theo dõi và cập nhật điểm số cho tất cả sân.

  • 5 sân độc lập
  • Mã PIN 4 chữ số
  • Hỗ trợ đơn & đôi
Phổ biến nhất

Điều khiển từ xa

Cập nhật điểm số từ bất kỳ thiết bị nào qua web browser. Không cần cài đặt, chỉ cần mã PIN.

  • Web responsive
  • Mobile friendly
  • QR Code access

Cập nhật real-time

Điểm số đồng bộ tức thì với SSE (Server-Sent Events). Khán giả theo dõi trận đấu mọi lúc, mọi nơi.

  • SSE streaming
  • Auto fallback
  • Tốc độ <80ms

Quản lý giải đấu

Tạo và quản lý giải đấu với đầy đủ nội dung thi đấu, đăng ký VĐV, lập lịch và xem kết quả.

  • Nhiều nội dung
  • Bốc thăm tự động
  • Lưu lịch sử

Cơ sở dữ liệu

Tích hợp SQL Server với JPA/Hibernate. Lưu trữ an toàn thông tin VĐV, CLB và kết quả.

  • SQL Server + H2
  • Remote access
  • Backup tự động

Giao diện hiện đại

Thiết kế đẹp mắt với FlatLaf theme và Bootstrap 5. Trải nghiệm mượt mà trên mọi thiết bị.

  • FlatLaf 3.4
  • Bootstrap 5.3
  • Dark mode
